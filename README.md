# skboy-site

Сайт-визитка для `skboy.vip`.

## Локально

Можно открыть `index.html` прямо в браузере. Для проверки серверного сохранения:

```powershell
cargo run
```

Админка: секретный адрес владельца.

Пароль задается переменной окружения `ADMIN_PASSWORD`.

## Render

Проект рассчитан на Render Web Service с Rust окружением. `render.yaml` уже содержит команду сборки и запуска.
